Conic Blitz

Description

The following is a fast paced game wherein the player is to answer a series of questions regarding conic sections and some important parts and measures. The player is expected to calculate the following:

Parabola Ellipse Hyperbola
Latus Rectum Length Latus Rectum Length Latus Rectum Length
Focus Coordinates Foci Coordinates Foci Coordinates
Vertex Coordinates Center Coordinates Center Coordinates
Eccentricity Eccentricity Eccentricity


Important Rules

Given a grand timer with 2 mins initial time, solve as many problems as possible! Each correct answer will merit you another 2 secs to solve. Each incorrect answer will demerit you 1 sec to solve.

Have fun!
